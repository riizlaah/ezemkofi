# Ezemkofi

Waktu pengerjaan sekitar 9 jam

## Features

- Basic Auth (login/register)
- View/Search coffee
- Add Coffee (with its size, e.g. S/M/L) to cart
- Remove a coffee from cart
- Change item quantity of a coffee in cart
- Using `SharedPreferences` to save Cart Items locally
- Checkout coffees
- Modern & interactive UI
